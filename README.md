# Erindi um gleðina við að leysa dæmi.
Þetta er stutt erindi fyrir úthlutun úr minningarsjóði Hjalta Þórs Ísleifssonar 3. janúar 2026

## Hvernig á að skoða?
Þessar glærur eru gerðar með rammanum [revealjs.com](https://revealjs.com/). Glærurnar má finna í skjalinu `index.html`.

Þið getið reynt að skoða glærurnar með því að keyra skipunina `npm start`. Þá ættuð þið að getað skoðað þetta í vafra undir slóðinni `localhost:8000`.

## Hvað næst?

- Fara yfir allt.
- Endurskipuleggja hirsluna.
